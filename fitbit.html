Fitbit Data Visualization

Area plot showing day by day walking data as collected by Fitbit Charge HR from September 2016 to December 2017. Select time frame on top left to resample data by a different interval.

Just a first run through to do some simple visualization of fitbit walking data over the last year+ using c3.js. Big thanks to gsx2json for an API that provides an amazingly simple way to connect to a public Google Drive spreadsheet and convert it to JSON data :D.

Next steps:

  • Incoporate other series like sleep data (daily data back to 2015), resting heart rate, 'number of active minutes', etc.
  • Bucket data by week and month so it's less noisy (partially complete)
